The Quality Lens workflow is configured to run on every push to the main branch and will:

1. Analyze code quality across 6 dimensions
2. Generate visual hexagon scores
3. Upload detailed artifacts
4. Provide actionable guidance for improvements

This ensures continuous quality monitoring for the Shannon penetration testing tool.
